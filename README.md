# TechRobot — Информационный сайт о технологиях и робототехнике

> **Дипломная (выпускная квалификационная) работа**  
> Стек: HTML5 · CSS3 · JavaScript ES6+ · JSON · localStorage  
> Без сторонних фреймворков

---

## Описание проекта

**TechRobot** — многостраничный информационный портал о современных технологиях и робототехнике. Сайт охватывает шесть ключевых направлений: искусственный интеллект, робототехника, интернет вещей, VR/AR, биотехнологии и беспилотные системы.

### Основные возможности

| Раздел | Функции |
|--------|---------|
| **Главная** | Hero с анимацией, категории, последние статьи, новости, превью каталога |
| **Статьи** | Сетка карточек, фильтр по 6 категориям, поиск, сортировка |
| **Статья** | Полный текст, автор, мета-данные, похожие статьи, «В избранное» |
| **Новости** | Лента с датами, тегами, разворачиваемым полным текстом |
| **Каталог роботов** | 12 роботов, фильтр по 6 типам, поиск, модалка с характеристиками |
| **Видеотека** | 8 YouTube iframe-плееров, фильтр по категориям |
| **Форум** | Темы и сообщения в localStorage, создание тем (для авторизованных) |
| **Блог** | Авторские записи с комментариями, разворачиваемый текст |
| **Поиск** | Живой поиск по статьям/новостям/роботам с подсветкой совпадений |
| **Личный кабинет** | Регистрация/вход, профиль, избранное, история, мои темы |
| **О нас / Контакты** | О проекте, команда, контакты Telegram/VK/Discord, форма |

## Как запустить

### Вариант 1 — Python (рекомендуется)

```bash
# Python 3
cd путь/к/проекту
python -m http.server 8080

# Откройте в браузере:
# http://localhost:8080
```

### Вариант 2 — Node.js

```bash
# Установите serve глобально (один раз)
npm install -g serve

# Запустите
serve . -p 8080
```

### Вариант 3 — VS Code Live Server

1. Установите расширение **Live Server**
2. Откройте папку проекта в VS Code
3. Правая кнопка на `index.html` → **Open with Live Server**

> ⚠️ **Важно:** Открывать через `file://` нельзя — браузеры блокируют fetch-запросы к локальным JSON-файлам из-за CORS. Нужен HTTP-сервер.

## Технические решения

### Авторизация (localStorage)

```
Ключи localStorage:
  tr_users          — массив пользователей (JSON)
  tr_current_user   — текущая сессия (JSON)
  tr_forum_topics   — темы форума (JSON)
```

**Хеширование пароля:** `btoa(btoa(SALT + password + SALT))`

> ⚠️ Это учебная реализация. В продакшене используйте серверную аутентификацию с bcrypt/argon2, HTTPS и httpOnly-куки.

### Данные

- Статические данные загружаются через `fetch('data/*.json')`
- Динамические данные (форум, избранное, история) — localStorage
- При первом запуске форум инициализируется из `data/forum.json`

### Компоненты

Шапка и подвал генерируются JavaScript-функциями (`components.js`) и вставляются в `<div id="header-placeholder">` на каждой странице — аналог include без серверного рендеринга.

### Поиск

Реализован на клиенте: все данные загружаются в память, поиск выполняется по строкам с `toLowerCase().includes()`. Результаты обновляются с дебаунсом 300 мс.

## Добавление изображений

Для замены заглушек на реальные изображения:

1. Поместите файлы в `images/articles/` и `images/robots/`
2. Имена файлов соответствуют полю `image` в JSON (например, `chatgpt.jpg`)
3. Рекомендуемый размер: статьи — 1200×675 px, роботы — 800×600 px
